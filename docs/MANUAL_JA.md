# マニュアル（日本語）— ORCA .hess → VEDA .fmu

## 1. このツールでできること
`orca_hess_to_veda_fmu.py` は ORCA の Hessian ファイル（`*.hess`）から、VEDA が読み込める
`*.fmu` を生成します。ここでいう `.fmu` は **Gaussian の fchk 互換“最小”テキスト**で、VEDA が
振動解析に必要とする情報（原子・座標・ヘッセ行列）を格納します。

本ツールは Gaussian の `.log`/`.out` を生成するものではありません。

必要に応じて `*.mpo`（結合情報の補助ファイル）も生成できます。

## 2. 入力
- 必須：`molecule.hess`
- 任意：`molecule.out`（電荷・多重度を取得できる場合があります）

## 3. 出力
- `molecule.fmu`（必ず生成）
- `molecule.mpo`（任意）

**`.mpo` について：**
- メイン変換ツールは通常 `molecule.mpo` を生成しますが、同名ファイルが存在する場合は上書きを避けるため `molecule_gen.mpo` を出力します。
- 結合情報を微調整したい場合（特に金属錯体など）は `adjust_mpo.py` / `adjust_mpo.pyw` を使用してください。
  - *調整モード*では `*_adj.mpo` として出力され、入力 `.mpo` は上書きしません。


入力と同じフォルダに出力されます。

## 4. 使い方
### GUI
- Windows：`orca_hess_to_veda_fmu.pyw` をダブルクリック
- あるいは：
  ```bash
  python orca_hess_to_veda_fmu.py
  ```

### VEDA
VEDA は `.fmu` / `.fmt` を要求します。
- 生成された `.fmu` を指定します。
- `.fmt` が必要な場合は、ORCA の `molecule.out` をコピーして `molecule.fmt` にリネームして
  同じフォルダに置く、などで対応できることがあります（ワークフロー依存）。

## 5. `.mpo`（結合情報）について
組み込みの `.mpo` 生成は共有結合半径ベースの距離判定（ヒューリスティック）です。
有機分子では十分なことが多い一方、金属錯体では調整が必要な場合があります。

結合が不自然な場合は：
- `adjust_mpo.py` を使って `.mpo` を再生成／修復する
- しきい値等を調整する（`adjust_mpo.py` のGUI）

## 6. トラブルシュート
- **VEDAが読み込まない**：パス、権限、`.fmu` が空でないか確認してください。
- **周波数が全体にズレる**：ヘッセ行列の単位や前提（最適化構造との整合）が崩れている可能性があります。
