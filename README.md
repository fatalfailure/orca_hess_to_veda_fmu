# ORCA Hessian to VEDA FMU Converter

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18377035.svg)](https://doi.org/10.5281/zenodo.18377035)

Convert an ORCA Hessian file (`.hess`) into a VEDA-readable `.fmu` file (minimal Gaussian *formatted checkpoint*-style text).
Optionally, generate an `.mpo` file (bond connectivity helper) to improve VEDA workflow.

---

## Features

- Convert ORCA `.hess` → VEDA `.fmu` (minimal fchk-style text)
- Optionally generate `.mpo` (connectivity helper)
- GUI launcher (`.pyw`) for easy use on Windows
- English and Japanese manuals included
- Example input/output files included (`examples/`)

---

## Requirements

- Python 3.x
- Standard library only (tkinter is included in most Python distributions)

> On some Linux distributions, `tkinter` may need to be installed separately.

---

## Installation

### Option A: Download from GitHub
1. Download this repository as a ZIP
2. Extract it
3. Run the script as described below

### Option B: Clone with git
```bash
git clone https://github.com/fatalfailure/orca_hess_to_veda_fmu.git
cd orca_hess_to_veda_fmu
```

---

## Usage (GUI)

### Main converter
- Windows: double-click  
  `orca_hess_to_veda_fmu.pyw`
- Or run with Python:
```bash
python orca_hess_to_veda_fmu.py
```

Steps:
1. Select your ORCA `.hess` file
2. (Optional) select your ORCA `.out` file (to read charge/multiplicity)
3. (Optional) enable `.mpo` generation
4. Click **Convert**
5. Output files will be created in the same folder:
   - `*.fmu` (always)
   - `*.mpo` (optional)

---

## Usage (MPO builder only)

If you only want to create or rebuild an `.mpo` file:
- Windows: `build_mpo.pyw`
- Or:
```bash
python build_mpo.py
```

---

## Output details

### `.fmu`
This file is a minimal Gaussian-like formatted checkpoint (fchk-style) text container used by VEDA.
It contains:
- Number of atoms
- Charge / Multiplicity (if available)
- Atomic numbers
- Cartesian coordinates
- Cartesian force constants (Hessian; packed lower triangle)

### `.mpo` (optional)
A connectivity helper file for VEDA to assist internal processing (e.g., DD2 construction).
It is generated by simple distance-based heuristics.
